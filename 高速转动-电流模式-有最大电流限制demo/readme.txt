当前设置成 CurrentLoopMode 后，速度环不参与控制，手动转动电机不会产生速度误差导致速度 PI 输出过大的反向电流。同时给定的 Iq_ref 很小，例如 ±0.30A，因此不容易触发过流保护关 PWM。

速度环不参与控制，控制链路就是：“你直接给 Iq_ref → 电流环 → PWM”

在电机静止的时候，手动转动电机不会因为速度误差过大而让速度 PI 输出过激电流。同时当前给定的 Iq 电流较小，因此不容易触发过流保护关 PWM。
